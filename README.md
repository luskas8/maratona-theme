Este é um tema de wordpress, feita para um teste de estágio na [Unisantos](https://www.unisantos.br).

## ⚙️ Instalação

Para usar este tema é bem simples, basta baixa-lo e extrai-lo na pasta de temas do WordPress.
Mas calma não tem nenhum dos dois instalados?

Se quiser um servidor local, recomendo usar uma dessas ferramentos, [XAMMP](https://www.apachefriends.org/pt_br/download.html) ou
o [LAMP](https://bitnami.com/stack/lamp/installer).

Agora chegou a hora de configurar o [WordPress](https://wordpress.org/download/). Baixe e coloque-o na pasta `htdocs` de onde você instalou o **LAMP** ou **XAMMP**. Feito isso crie uma tabela no banco de dados, `MySQL` chamada wordpress, entre em [localhost:8080/wordpress](https://localhost:8080/wordpress) e
siga o passo a passo. Após a instalação pronta, entre em [https://localhost:8080/wordpress/wp-admin](https://localhost:8080/wordpress/wp-admin) logue com suas credênciais, válidas!
Indo na aba **Aparência** e depois **Temas**, mude o tema para o que foi baixado, ou caso ainda não tenha-o colocado na pasta correta, adicione o arquivo e depois ative-o.

Pronto agora é só ir na página do [localhost:8080/wordpress](https://localhost:8080/wordpress) e estará vendo o layout!! 😁😁
